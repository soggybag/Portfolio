Tetris Redux App

This is an implementation of the arcade game Tetris. This version of the game was built using React and Redux. The game board is made up of a two dimensional array where the block falling down the screen is represented by another two dimensional array mapped onto the first.

I made this an open source project for Hacktoberfest and had a few contributions.

Course Book

Course book is a web app for tracking students in class. The goal was to have an app that would choose random students during class for questions and answers, and randomly pair students for activities. When Choosing students Course Book works through all students until everyone has been picked before any names are repeated. Course Book supports multiple classes.

You view Course Book live on the web here. Course Book saves all data to local storage. There is no log in or sign up. Instead all data you enter is saved to the computer or mobile device you are using. This has a couple consequences. First, the app is simple and easy to use. It also works offline. Second, if you enter data on your desktop it _isn't_ available on your phone or laptop.

Simple Journal

Simple Journal is a new take on the todo list. It allows you to create todo lists and mark items in the list complete. When creating a new list items from the previous list that are not complete are move to the new list. The idea is to create daily todo lists where items that weren't marked complete get carried over.

This app was built with React and Redux. All of the data is stored with local storage. This makes an easy quick app. It does have the limitation that data doesn't follow you from device to device or browser to browser. If you make a list in Safari on your laptop that list will remain in Safari on your laptop.
